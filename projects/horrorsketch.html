Year Long Freedom Project #2

For this challenge, I worked on using the different concepts of Javascript and the basics of EarSketch to make a song that you would usually hear in a horror based game or movie. I used concepts such as conditionals where I used `if` statments and `for` loops to create certain affects to make it so that people would be interested in the song that I had made.

A problem that I faced was figuring out how exactly to integrate Javascript with EarSketch since Earsketch has a wide variety of sounds to choose from that I could have also just placed into certain spots and run my code. I solved this issue by looking at the curriculum which goes over the different parts of how to use EarSketch. By doing this, I get to use some of the knowledge that I have gained from learning Javascript into my Year Long Project.
